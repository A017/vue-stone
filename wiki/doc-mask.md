# 遮罩组件

## Usage

```html
<v-mask></v-mask>
```

## Options

属性名   |    类型    |    默认值    |   说明
----    | ----    | ----    | ----    |
v-model |  Boolean  | false | 是否显示mask 。
transition | String | 'fade' | 更改 vue transition 的 name 属性值

## Events
事件名称   |    说明    |    参数
----    | ----    | ----
click |  点击mask时会触发该事件  | 无
change | mask 组件切换显隐状态时会触发此事件 | value
