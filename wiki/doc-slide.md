# Slide

> 本组件库中的轮播组件只提供了最基本的功能，如果需要更强大的 vue 轮播组件的话，请关注 [c-swipe](https://github.com/pspgbhu/Vue2-C-Swipe-Mobile)

## Tag Name
`c-slide`

## Options
属性名   |    类型    |    默认值    |   说明
----    | ----      | ----        | ----    |
v-model | Number | 0 | 当前激活状态下的卡片的索引
pagination | Boolean | true | 是否显示导航器



## Events
事件名   |    说明    |   参数
----    | ----      | ----        |
change  | 卡片切换时触发此事件 | 当前激活的卡片的索引值
